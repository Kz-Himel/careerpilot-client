# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/your-username/careerpilot-client.git
```

Go to Project

```bash
cd careerpilot-client
```

Install Packages

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Build Production

```bash
npm run build
```
